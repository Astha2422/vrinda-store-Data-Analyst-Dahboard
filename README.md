📊 Vrinda Store Annual Report Dashboard (Excel)
📌 Overview

This project is an interactive Excel dashboard created to analyze the annual performance of Vrinda Store (2026).
It provides insights into sales, orders, customer demographics, and regional performance using dynamic charts and slicers.

🚀 Features
📈 Order vs Sales Analysis
👨‍🦱👩‍🦰 Gender-wise Sales & Orders Distribution
📦 Order Status Tracking
🌍 Top 5 States by Sales
🛒 Sales Channel Analysis (Amazon, Flipkart, Ajio, etc.)
🎛️ Interactive Filters (Category, Channel, Month)
🛠️ Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Data Cleaning Techniques
