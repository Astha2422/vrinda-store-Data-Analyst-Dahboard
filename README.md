📊 Vrinda Store Annual Report Dashboard (Excel)
📌 Overview

This project is an interactive Excel dashboard created to analyze the annual performance of Vrinda Store (2026).
It provides insights into sales, orders, customer demographics, and regional performance using dynamic charts and slicers.
 Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Slicers & Filters
Data Cleaning Techniques
